Alert

Alerts are used to communicate a state that affects a system, feature or page.

Import

Trend Micro Styled UI exports 3 alert related components.

  • Alert: The wrapper for alert components.
  • AlertIcon: The visual icon for the alert that changes based on the status prop.
import {
  Alert,
  AlertIcon,
  AlertButton,
} from '@trendmicro/react-styled-ui/Alert';

Usage

Your browser is outdated!
Your XDR experience may be degraded.
<Alert status="error">
  <Flex width="100%" justifyContent="space-between">
    <Flex alignItems="center">
      <AlertIcon />
      <Heading mr="2x" fontWeight="bold">Your browser is outdated!</Heading>
      <Text>Your XDR experience may be degraded.</Text>
    </Flex>
    <Flex>
      <AlertButton minWidth={76}>Action</AlertButton>
      <Space width="1rem" />
      <ButtonBase color="black:secondary">X</ButtonBase>
    </Flex>
  </Flex>
</Alert>

Status

Change the status of the alerts by passing the status prop. This affects the color scheme and icon used. Alert supports error, warning, and info statuses.

There was an error processing your request
Seems your account is about to expire, upgrade now!
XDR is going live on August 30th. Get ready!
<Stack spacing=".5rem" shouldWrapChildren>
  <Alert status="error">
    <AlertIcon />
    There was an error processing your request
  </Alert>
  <Alert status="warning">
    <AlertIcon />
    Seems your account is about to expire, upgrade now!
  </Alert>
  <Alert status="info">
    <AlertIcon />
    XDR is going live on August 30th. Get ready!
  </Alert>
</Stack>

Composition

Alert ships with smaller component to allow for flexibility and make it easy to create all kinds of layout. Here's an example of custom alert style and layout.

Application submitted!
Thanks for submitting your application. Our team will get back to you soon.
<Alert
  status="info"
  flexDirection="column"
  justifyContent="center"
  textAlign="center"
  height="200px"
>
  <AlertIcon size={40} mr={0} />
  <Heading mt={4} mb={1} fontSize="lg" fontWeight="bold">
    Application submitted!
  </Heading>
  <Text maxWidth="sm">
    Thanks for submitting your application. Our team will get back to you soon.
  </Text>
</Alert>

Props

Alert props

Alert is the wrapper for alert component. It composes the Flex component.

NameTypeDefaultDescription
statusstring'info'The status of the alert. One of: 'error', 'warning', 'info'

AlertIcon props

AlertIcon composes Icon and changes the icon based on the status prop.

AlertButton props

AlertButton composes the PseudoBox component.

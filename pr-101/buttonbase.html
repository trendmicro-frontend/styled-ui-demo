ButtonBase

ButtonBase does not have appearance settings including default border, color, outline, and padding. It aims to be a simple building block for creating a button.

Import

import ButtonBase from '@trendmicro/react-styled-core/ButtonBase';
// or
import { ButtonBase } from '@trendmicro/react-styled-core';

Usage

<ButtonBase>Click Me</ButtonBase>
EDITABLE EXAMPLE

Attributes

Standard button attributes are supported, e.g., type, disabled, etc.

disabled

<ButtonBase disabled>Disabled</ButtonBase>
EDITABLE EXAMPLE

Plain button

function Example() {
  const [value, setValue] = React.useState(0);
  const handleClear = () => setValue(0);
  const handleAdd = () => setValue(value => value + 1);
  return (
    <Stack direction="column" alignItems="flex-start">
      <ButtonBase onClick={handleClear}>
        <Badge>{value}</Badge>
      </ButtonBase>
      <Space height="1rem" />
      <ButtonBase
        opacity={.6}
        _hover={{ opacity: .8 }}
        onClick={handleAdd}
      >
        Click Me
      </ButtonBase>
    </Stack>
  );
}

Adding an icon to a plain button

0
function Example() {
  const [value, setValue] = React.useState(false);
  const handleClick = () => {
    setValue(value => !value);
  };
  const btnStyleProps = {
    color: value > 0 ? 'blue:50' : 'inherit',
    opacity: value > 0 ? 1 : .6,
    _active: {
      opacity: 1,
    },
    _hover: {
      opacity: .8,
    },
  };
  return (
    <Stack direction="column" alignItems="flex-start">
      <Badge>{value ? '1' : '0'}</Badge>
      <Space height="1rem" />
      <ButtonBase
        onClick={handleClick}
        {...btnStyleProps}
      >
        <FontAwesomeIcon icon="thumbs-up" fixedWidth style={{ color: 'inherit' }} />
        <Space width=".25rem" />
        Like
      </ButtonBase>
    </Stack>
  );
}

Props

ButtonBase composes PseudoBox. You can override the default styles with style props.

NameTypeDefaultDescription
disabledbooleanIf true, the button will be disabled. This sets aria-disabled=true and you can style this state by passing the _disabled prop.

UTILITIES

Getting Started

Install Tonic Styled UI and its peer dependencies.

npm install @trendmicro/react-styled-ui

Create a theme object to include custom color, typography, and layout values. Styled UI comes with a default theme, so this step is optional.

// example base.js
export default {
  fonts: {
    base: '-apple-system, BlinkMacSystemFont, "Segoe UI", "Helvetica Neue", Helvetica, Arial, sans-serif',
    mono: '"SFMono-Medium", "SF Mono", "Segoe UI Mono", Menlo, Consolas, Courier, monospace',
  };
  fontSizes: {
    xs: '.75rem',
    sm: '.875rem',
    md: '1rem',
    lg: '1.125rem',
    xl: '1.25rem',
    '2xl': '1.5rem',
    '3xl': '1.75rem',
    '4xl': '2rem',
  },
};

You can also extend the theme to add your custom colors, fonts, etc.

import { theme } from '@trendmicro/react-styled-ui';


// Let's say you want to add custom colors
const customTheme = {
  ...theme,
  colors: {
    ...theme.colors,
    brand: {
      90: "#1a365d",
      80: "#153e75",
      70: "#2a69ac",
    },
  },
};

Add the theme to your application with the ThemeProvider, passing in the theme object as a prop.

import React from 'react';
import { ThemeProvider } from '@trendmicro/react-styled-ui';


// Use at the root of your app
<ThemeProvider theme={customTheme}>{props.children}</ThemeProvider>;

Using components

Import components using named imports from @trendmicro/react-styled-ui, as shown below:

import { Button } from '@trendmicro/react-styled-ui';

Recommended setup

Sometimes you may need to apply base CSS styles to your application. Styled UI export a CSSBaseline that will normalize styles for a wide range of elements, which is based on the normalize.css.

CSSBaseline is recommended to add at the root to ensure all components work correctly.

import { css, Global } from '@emotion/react';
import {
  Box,
  ColorModeProvider,
  ColorStyleProvider,
  CSSBaseline,
  ThemeProvider,
  useColorMode,
  useTheme,
} from '@trendmicro/react-styled-ui';


import App from './App';


const Layout = (props) => {
  const [colorMode] = useColorMode();
  const [colorStyle] = useColorStyle({ colorMode });
  const { fontSizes, lineHeights } = useTheme();
  const backgroundColor = colorStyle.background.primary;
  const color = colorStyle.text.primary;


  return (
    <>
      <Global
        styles={css`
          body {
            font-size: ${fontSizes.sm};
            line-height: ${lineHeights.sm};
          }
        `}
      />
      <Box
        backgroundColor={backgroundColor}
        color={color}
        fontSize="sm"
        lineHeight="sm"
        height="100vh"
        {...props}
      />
    </>
  );
};


const CustomApp = (props) => {
  return (
    <ThemeProvider>
      <ColorModeProvider value="dark">
        <ColorStyleProvider>
          <CSSBaseline />
          <Layout>
            <App {...props} />
          </Layout>
        </ColorStyleProvider>
      </ColorModeProvider>
    </ThemeProvider>
  );
};


export default CustomApp;

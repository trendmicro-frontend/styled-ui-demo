Style Props

Style props is a way to use CSS-in-JS styling technique to pass styles as props to style components without having to use a separate CSS file.

Reference Table

The followings are the reference table for style props. Each style prop corresponds to at least one CSS property, and some of them map to scales defined in the theme.
